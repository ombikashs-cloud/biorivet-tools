# ANTIGRAVITY PROMPT — Tool 1: NEET Diagnostic Tool Build a single HTML file diagnostic tool for NEET institutes. 3-step flow: Step 1 — Upload Excel with: Q No, Question, Chapter, Sub-Concept, Error Type, Answer (A/B/C/D) Step 2 — Upload CSV/Excel with: Student Name, Q1, Q2...Qn answers (A/B/C/D) Step 3 — Show diagnostic report with 3 sections: Section A: Chapter-wise error rate table with progress bars → columns: Chapter | Questions | Total Wrong | Max Possible | Error Rate % | Bar | Weakness Badge → badge logic: ≥60% = Critical Gap (red), ≥40% = Needs Attention (yellow), <40% = Strong (green) Section B: Per-student table. Click to expand each student. → shows: score, weakest chapter, 2nd weakest, priority badge → expanded: answer grid (green=correct, red=wrong), chapter breakdown, 3-day action plan Section C: 5 auto-generated batch insight bullets Export button: generates 4-sheet Excel report using SheetJS Tech: Vanilla HTML+CSS+JS only. SheetJS CDN. Single self-contained file. No frameworks. Design: Navy (#1F3864) + white. BioRivet footer. Mobile responsive. Drag-and-drop upload.
# ANTIGRAVITY PROMPT — Tool 2: Student Dropout Risk Tracker Build a single HTML file dropout risk prediction tool for NEET institutes. INPUT: Excel upload with columns: Student Name | Attendance % | Test 1 Score | Test 2 Score | Test 3 Score | Fee Status (Paid/Pending) | Counselling Sessions PROCESSING: Risk Score (0–100) = (100 - Attendance%) × 0.30 + Score Decline Weight × 0.40 + Fee Penalty × 0.20 + (10 - min(Counselling,10)) × 0.10 Score decline = compare Test 3 vs Test 1 percentage drop OUTPUT DASHBOARD: → 3 metric cards: Total High Risk | Average Attendance | Students with Fee Pending → Sortable table: Name | Attendance | Avg Score | Score Trend | Fee | Risk Score | Risk Badge → Top 5 at-risk students highlighted in red panel with intervention note per student → Risk badge: ≥70 = High Risk (red), ≥40 = Watch (yellow), <40 = Safe (green) INTERVENTIONS (auto-generated per student): High Risk → "Schedule immediate counselling. Check fee status. Call parents this week." Watch → "Monitor attendance next 2 weeks. Assign peer buddy." Safe → "Continue regular engagement." Export: One-click Excel export of full risk report Tech: Vanilla HTML+CSS+JS. SheetJS CDN. Single file. Navy + orange design. BioRivet branding.
# ANTIGRAVITY PROMPT — Tool 3: Test Paper Difficulty Analyser Build a single HTML file test paper quality audit tool. INPUT: Excel upload with columns: Q No | Question | Chapter | Difficulty (Easy/Medium/Hard) | Marks | Answer ANALYSIS: → Overall difficulty distribution: count + % of Easy/Medium/Hard → Chapter-wise difficulty table: per chapter show Easy/Medium/Hard counts → Paper balance score (0–100): ideal = 20% Easy, 50% Medium, 30% Hard → Identify chapters that are over-tested or under-tested → Estimated average student score (assume 90% on Easy, 60% on Medium, 30% on Hard) OUTPUT: → Summary cards: Total Questions | Paper Balance Score | Estimated Avg Score | Most Tested Chapter → Difficulty donut chart (CSS-only, no chart library) → Chapter analysis table → Recommendations panel: - If Easy > 40%: "Paper is too easy. Increase Hard questions." - If Hard > 50%: "Paper too difficult. Students may demotivate." - If a chapter has 0 questions: "Chapter X not covered — check syllabus alignment." Tech: Vanilla HTML+CSS+JS. SheetJS CDN. Single file. Navy + purple design. BioRivet branding.
# ANTIGRAVITY PROMPT — Tool 4: NEET Score Predictor Build a single HTML file NEET score prediction tool for students. INPUT FORM (no file upload — student fills in directly): → 5 sliders (0–100): Physics score | Chemistry score | Botany score | Zoology score | Revision cycles (0–10) → Number input: Daily study hours (1–12) → Dropdown: Months to NEET exam (1–12) → Checkbox: Have you completed previous year papers? (Y/N) SCORE FORMULA: Raw = (Physics × 0.25) + (Chemistry × 0.25) + (Botany × 0.25) + (Zoology × 0.25) Boost = (RevisionCycles × 2) + (StudyHours × 1.5) + (MonthsLeft × 0.5) + (PYQ × 10) Predicted NEET = min(720, Raw × 7.2 + Boost) Range: ±30 marks OUTPUT: → Predicted score range (e.g. 540–600 / 720) → Percentile estimate (use lookup table: 600=99%, 550=97%, 500=94%, 450=88%...) → Strongest subject badge + Weakest subject badge → 4-point personalised focus plan based on lowest subject slider → Motivational message based on score range Design: Student-friendly, colorful but clean. Mobile-first. BioRivet branding. No upload needed. Tech: Vanilla HTML+CSS+JS only. Single file.
# ANTIGRAVITY PROMPT — Tool 5: Batch Comparison Tool Build a single HTML file batch comparison tool for NEET institute owners. INPUT: → Upload Batch A: Excel file (same format as Diagnostic Tool — Student Name + Q1...Qn answers + Question Paper) → Upload Batch B: Same format → Both batches must use the same question paper COMPARISON METRICS: → Average score per batch → High risk student % per batch → Chapter-wise error rate per batch (side-by-side) → Score distribution: how many students in 0–30%, 30–50%, 50–70%, 70–100% range OUTPUT DASHBOARD: → Header cards: Batch A students | Batch B students | Winner (higher avg score) → Chapter comparison table: Chapter | Batch A Error % | Batch B Error % | Winner badge → Score distribution bars side by side per batch → Summary: "Batch A stronger in X chapters. Batch B stronger in Y chapters. Batch B has more high-risk students — prioritise dropout intervention." Tech: Vanilla HTML+CSS+JS. SheetJS CDN. Single file. Navy + red design. BioRivet branding.
