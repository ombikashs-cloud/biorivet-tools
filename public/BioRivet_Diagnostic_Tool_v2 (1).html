📋 Question Paper Format (Excel)
Required columns (exact header names — case insensitive):
Q No  | Question  | Chapter  | Sub-Concept  | Error Type  | Answer
Q1    | Which enzyme... | Digestion | Enzyme Activation | Memory | B
Q2    | Chylomicrons... | Digestion | Lipid Absorption | Memory | A
Q3    | Normal BP... | Circulation | Blood Pressure | Memory | C
  • Q No — Question number (Q1, Q2... or 1, 2...)
  • Question — The actual text of the question (Recommended)
  • Chapter — Used to group questions (e.g. Digestion, Circulation)
  • Error Type — Memory / Conceptual / Application
  • Answer — Correct answer: A, B, C, or D
  • ✨ AI Auto-Tagging: If you only upload the Question text, BioRivet AI can automatically infer the Chapter, Error Type, and Correct Answer!
